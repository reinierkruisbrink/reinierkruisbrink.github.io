# On Learning

Learning is a lifelong journey that requires patience, curiosity, and humility.

[[index|Essays]] | [[On Simplicity]]

## Key Insights

The best learning happens when we embrace discomfort and uncertainty. It's in the struggle that understanding deepens.

## Practices

- **Active recall**: Test yourself rather than passively reviewing
- **Spaced repetition**: Distribute practice over time
- **Teaching others**: Solidify knowledge by explaining it
