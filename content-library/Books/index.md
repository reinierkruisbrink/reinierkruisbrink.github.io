# Books

A collection of books I've read, annotated with key concepts, connections, and personal reflections.

## Tag Taxonomy

Tags are organized into three layers: **Broad Domains** that map to major fields of knowledge, **Semi-Specific Domains** that zoom into particular disciplines, and **Cross-Cutting Themes** that bridge multiple domains.

---

### Broad Domains

High-level fields of knowledge. Each book typically touches one or more of these.

| Tag | Description | Example Topics |
|---|---|---|
| `biology` | Life sciences broadly | Organisms, cells, life processes, biodiversity |
| `neuroscience` | Brain, nervous system, neural mechanisms | Neurons, brain regions, neuroplasticity, neural signaling |
| `mathematics` | Mathematical thinking, modeling, formal frameworks | Equations, proofs, geometry, mathematical structures |
| `philosophy` | Fundamental questions, ethics, epistemology | Mind–body problem, free will, morality, knowledge |
| `society` | Social structures, policy, politics, culture | Governance, inequality, cultural norms, collective behavior |
| `environmental-science` | Earth systems, climate, pollution, resource management | Carbon cycle, deforestation, pollution, planetary boundaries |
| `physics` | Physical laws, forces, matter/energy | Thermodynamics, mechanics, electromagnetism, relativity |
| `history` | Historical events, civilizations, cultural development | Empires, revolutions, trade routes, archaeological record |
| `linguistics` | Language structure, meaning, origins, etymology | Syntax, semantics, language families, word origins |
| `classics` | Ancient literature, epic poetry, retellings | Greek myth, Roman epic, oral tradition, mythological archetypes |
| `fiction` | Novels, literary narratives | Allegory, magical realism, dystopia, coming-of-age |
